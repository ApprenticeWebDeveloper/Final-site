Klondike what is it?

Klondike is our open to everyone Java/Bedrock crossplay server! With this server you will
be able to join your friends on any platform.To accomplish this Me and Wingsoffire123 have
installed spigot a minecraft software allowing an extension called geyser to be added on.
Geyser allows Java and Bedock players to join the same server but does not allow clientside mods.
We will try to install all quality of life mods on serverside but it is possible it will be completely
vanilla. Either way a great way to connect with friends! There is no additional mod download required.
Just run your regular client on the most recent version of Minecraft.
